# Plantagotchi 🌱

앱인토스(Toss 미니앱)용 가상 식물 키우기 게임입니다.

## 게임 소개

다마고치 스타일로 매일 가상 식물을 돌보며 성장시키는 미니앱입니다.
매일 미션을 수행하고 식물을 케어하면 씨앗에서 황금 식물까지 성장합니다.

## 식물 단계

🌰 씨앗 → 🌱 새싹 → 🌿 어린 식물 → 🌸 꽃봉오리 → 🌺 만개 → ✨🌟 황금 식물

## 주요 기능

- 매일 3가지 랜덤 미션 (날짜 기반 시드로 일관성 유지)
- 수분/햇빛/건강 3가지 스탯 관리
- 시간 경과에 따른 스탯 감소 (방치 패널티)
- 광고 시청 시 스탯 부스트
- 다크모드 지원
- localStorage 기반 상태 영속성

## 기술 스택

- Next.js 16 + TypeScript
- Tailwind CSS v4
- @apps-in-toss/web-framework ^1.14.1
- @toss/tds-mobile ^2.2.1
- @emotion/react + @emotion/styled
- React 19
